# DATA CARD — `ml_engine` synthetic dataset (Track B)

> **EXPERIMENTAL · 100% SYNTHETIC · NO real patients · NO clinical validity.**

## What it is
Dataset generated by `ml_engine.core.synth.generate` from per-diagnosis
priors (`ml_engine.domains.vertigo.SYNTHETIC`). Each row = one synthetic case
with de-identified features + a `label` (one of the 8 trainable diagnoses).
**It does not come from any patient, medical record, or real registry.** No
PII of any kind is involved (the data is fabricated by sampling).

## How it is generated
- **Conditional sampling per label:** for each diagnosis, provisional feature
  distributions derived from the ICVD criteria (categoricals via probability
  table, booleans via prevalence, numerics via truncated normal). Features not
  specified by a label → neutral default.
- **Prevalence:** per label (provisional, documented in `vertigo.SYNTHETIC`),
  not uniform.
- **Size / seed:** `n_samples=8000`, `seed=20260711` (fixed). Downstream
  splits: train/calibration/test (60/20/20), all synthetic.
- **Reproducibility:** same seed ⇒ same DataFrame **under the locked
  environment** (`requirements.lock`) and with numeric tolerance; byte-for-byte
  identity across platforms/threads is NOT promised.

## What it is NOT (honesty limits)
- **It is not clinical evidence.** Metrics computed on this data (F1, AUC,
  ECE, calibration) measure **recovery of the synthetic generative process** —
  how well the model reconstructs the rules the data was fabricated with —
  **not** diagnostic performance on patients.
- **The priors are PROVISIONAL.** Prevalences, distributions, and thresholds
  must be reviewed/signed by the specialist (T-CLIN). Until then: "prototype
  assumptions, pending clinical review".
- **Known circularity risk:** the label generates the features and the model
  re-learns them; high metrics do NOT imply clinical utility. That is why they
  are reported as generator recovery and accompanied by stress sets.

## Intended use
Demonstrating the **mechanism** of the confidence layer (hierarchy, danger
monotonicity, calibration, abstention, SHAP) in a reproducible and honest way.
Not for clinical inference.
